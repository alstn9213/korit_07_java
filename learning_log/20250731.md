# while 반복문

```declarative
while() {
    
}
```
조건식이 true일 때, 실행부분이 반복적으로 이루어진다.  
그래서 while 내부의 조건식이 특정 시점에 false가 되도록 설계해둘 필요가 있다. 그렇지 않으면 무한 루프 발생한다.  
while은 if문 처럼 중첩이 가능하고 while과 if를 같이 쓰는 것도 가능하다.

중첩 while문(Nested-while)
```declarative
while(){
    
    while() {
     
    }
    
}
```

scope(범위): 전역변수/지역변수
